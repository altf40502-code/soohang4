# soohang4
수행평가4_21124 허정욱

Unity와 C#을 기반으로 제작한 쿼터뷰 액션 RPG 프로젝트
플레이어는 몬스터를 처치하고 퀘스트를 진행하며 성장하고, 외부 AI API와 연동된 AI NPC와 대화하며 게임 스토리를 진행할 수 있다. 이 프로젝트는 단순한 NPC 대화 시스템이 아니라, 외부 AI API를 활용하여 플레이어의 질문에 능동적으로 답변하는 AI NPC를 구현하는 것을 목표로 한다. AI NPC는 게임 세계관과 규칙에 맞춰 답변하며, 추후 퀘스트, 아이템, 길 안내 등 다양한 게임 시스템과 연동할 수 있도록 설계되었다. 

<조작 방법>
WASD	플레이어 이동
Space	점프
Left Ctrl	회피
/	AI 채팅창 열기
ESC	AI 채팅창 닫기
Enter	질문 전송
